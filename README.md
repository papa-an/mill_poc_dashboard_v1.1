# Palm Oil Mill Performance Dashboard

A static dashboard for analyzing Palm Oil Mill performance metrics like OER and Fruit Mix.

## Features
- **OER Analysis**: Pre-LMM vs Post-LMM comparison.
- **Fruit Mix**: Inti, Plasma, and 3rd Party analysis.
- **Interactive Charts**: Trend lines and bar charts.
- **Filtering**: By Mill, PSM, LMM status, and Date.

## Tech Stack
- HTML5 / TailwindCSS
- Vanilla JavaScript
- Chart.js
- SheetJS (xlsx)
